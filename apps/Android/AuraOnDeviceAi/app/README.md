# AuraOnDevice-AI: Mobile App 📱

AuraOnDevice-AI is a state-of-the-art Android application that brings advanced AI models directly to your pocket. By leveraging industry-leading inference engines, we provide a private, fast, and feature-rich environment for on-device intelligence.

## 🚀 What's New: Gemma 4 Support
We are excited to announce support for the latest **Gemma 4** family of models! Using our optimized **LiteRT-LM** engine, these models deliver unprecedented performance and accuracy on mobile hardware. 

- **Lightning Fast**: Optimized for modern Android NPUs and GPUs.
- **Privacy First**: All inference happens locally—no data ever leaves your device.
- **Advanced Capabilities**: From complex reasoning to creative writing, Gemma 4 is now fully accessible on-the-go.

## 🏗️ Core Technologies
- **MNN-LLM**: High-performance backend for Qwen, Llama, and DeepSeek models.
- **MediaPipe**: Seamless integration for Gemma 3 and specialized AI Studio tasks.
- **LiteRT-LM**: The modern stack for Gemma 4 and experimental vision-language models.
